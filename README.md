# Gen-Z Culture meets Christhood

A simple static event website scaffold.

## What's included

- `home.html` — homepage with hero, schedule, speakers, and RSVP form
- `css/styles.css` — responsive styles
- `js/main.js` — client-side RSVP handling (stores emails in localStorage)

## Preview locally

Open `home.html` in your browser or run a local server:

```bash
# using Python 3
python -m http.server 8000
# then open http://localhost:8000/home.html
```
